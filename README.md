# Pascal CodeX 1.0

![PascalCodeX](logo.ico)

**Профессиональный редактор кода для Pascal с компиляцией и запуском**

---

## О программе

Pascal CodeX — это легкий и мощный редактор кода, созданный специально для языка Pascal. Сочетает простоту Блокнота с возможностями профессиональных сред разработки. Подсветка синтаксиса, встроенная компиляция через Free Pascal Compiler, запуск программ и многоязычный интерфейс делают его удобным инструментом для обучения и повседневной работы.

## Возможности

### Подсветка синтаксиса Pascal
Полная подсветка синтаксиса языка Pascal. Ключевые слова, типы данных, строки, комментарии, числа и директивы компилятора отображаются разными цветами. Поддерживаются все виды комментариев: однострочные, многострочные в фигурных скобках и многострочные в звездочках.

### Встроенная компиляция и запуск
Компилируйте и запускайте программы прямо из редактора, не открывая командную строку. Компилятор Free Pascal встроен в установщик. Панель вывода показывает процесс компиляции, ошибки и результат выполнения программы в реальном времени. Остановить выполнение можно в любой момент.

### Многоязычный интерфейс
Поддержка трех языков интерфейса:
- Русский
- Английский
- Карачаевский

Язык переключается в настройках и сохраняется автоматически при следующем запуске.

### Нумерация строк
Каждая строка пронумерована для удобной навигации. Текущая строка подсвечивается.

### Автоотступы
Редактор автоматически сохраняет уровень отступа при создании новой строки. После ключевых слов begin, var, type, const, record, then, else, do, repeat отступ автоматически увеличивается.

### Работа с выделенным текстом
Выделите несколько строк и нажмите Tab для добавления отступов. Shift и Tab убирает отступы. Выделение сохраняется после каждой операции.

### Управление шрифтом
Изменяйте размер шрифта через Ctrl и колесико мыши или через меню. В настройках можно выбрать любой моноширинный шрифт, установленный в системе.

### Панель вывода
Результаты компиляции и выполнения программ отображаются в отдельной панели под редактором. Ошибки выделяются красным цветом. Очистка панели производится одной кнопкой.

### Справочник синтаксиса Pascal
В меню Справка доступен полный справочник синтаксиса с примерами: структура программы, переменные, константы, типы данных, ввод и вывод, условия, циклы, процедуры, функции, массивы, записи, множества, файлы, указатели и модули.

### Темная тема
Профессиональная темная цветовая схема для комфортной работы в течение длительного времени.

## Системные требования

- Windows 7, 8, 10 или 11 (32-бит или 64-бит)
- 2 ГБ оперативной памяти
- 300 МБ свободного места на диске

## Установка

### Установщик
Скачайте установщик `PascalCodeX_Setup.exe`, запустите и следуйте инструкциям. Программа установится со всеми необходимыми файлами, включая компилятор Free Pascal.

### Портативная версия
Скачайте архив, распакуйте в любое удобное место и запустите `PascalCodeX.exe`. Для компиляции программ разместите папку `pascal` с компилятором рядом с исполняемым файлом.

## Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| Ctrl и N | Новый файл |
| Ctrl и O | Открыть файл |
| Ctrl и S | Сохранить файл |
| Ctrl и Shift и S | Сохранить файл как |
| Ctrl и Z | Отменить действие |
| Ctrl и Y | Повторить действие |
| Ctrl и F | Найти и заменить |
| F5 | Компилировать и запустить |
| Ctrl и C | Остановить выполнение |
| F12 | Показать или скрыть панель вывода |
| Ctrl и Плюс | Увеличить шрифт |
| Ctrl и Минус | Уменьшить шрифт |
| Ctrl и колесико мыши | Изменить размер шрифта |
| Tab | Добавить отступ |
| Shift и Tab | Убрать отступ |
| Alt и F4 | Выйти из приложения |
